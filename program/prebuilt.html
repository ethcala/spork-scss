Carousel

The carousel above uses the following code. The prev/next buttons, captions and markers can be left out if desired. If you want the carousel to move automatically, include 'carousel-auto' along with 'carousel-holder' (if not, don't include carousel-auto). You will also need to include a data-time=[any time] attribute. The example uses 5000. This represents the amount of time between slide movement.

This carousel uses the fade animation, which must be placed as a class in every slides div. The rect markers are used as indicators and need to have the same amount as there are slides. Images should have the 'no-img-style' class to prevent the default image styling. All unspecified elements, such as w-100 on images and the onclick attributes, must be included.

<div class='carousel-holder carousel-auto' data-time='5000'>

  <div class='slides fade'>

    <img src='https://via.placeholder.com/600x150?text=Image+1' class='no-img-style w-100'>

    <div class='caption'>Caption 1</div>

  </div>

  <div class='slides fade'>

    <img src='https://via.placeholder.com/600x150?text=Image+2' class='no-img-style w-100'>

    <div class='caption'>Caption 2</div>

  </div>

  <div class='slides fade'>

    <img src='https://via.placeholder.com/600x150?text=Image+3' class='no-img-style w-100'>

    <div class='caption'>Caption 3</div>

  </div>

  <a class='prev' onclick='moveSlides(-1)'>❮</a>

  <a class='next' onclick='moveSlides(1)'>❯</a>

</div>

<div class='txt-a-center rect-holder'>

  <span class='rect-marker' onclick='currentSlide(1)'></span>

  <span class='rect-marker' onclick='currentSlide(2)'></span>

  <span class='rect-marker' onclick='currentSlide(3)'></span>

</div>

Prebuilt Buttons

These are the check, cancel and info buttons (classes called 'check-button', 'x-button' and 'q-button'). They are created with a div given the correct class and any text tag inside with capital X, ?, or the check HTML code, although you may put in whatever you want. They both include a transition in colour and size when moused over. They are 30x30 px by default.

The transitions can be edited and applied to other elements by importing 'mixins' and using @include 'button-change(color, width in px, height in px)' in the element. Any colour and size can be supplied.

Pagination

To create pagination buttons, create a div with the row class and nest any amount of divs inside of it. The first should be 'page-btn-start(-scnd)', the middle ones should be 'page-btn(-scnd)', and the last should be 'page-btn-end(-scnd)'. The default is a darker version of the main colour, but the -scnd ones are a version of the secondary colour.

<div class='row'>

  <div class='page-btn-start-scnd'>1</div>

  <div class='page-btn-scnd'>2</div>

  <div class='page-btn-end-scnd'>3</div>

</div>

Badges

Badges are added to text by putting both in a row div and adding text and then a div with the correct badge class. All are named 'badge-[color in example]', except the regular one which is just 'badge'. To create the pill or circle shape (circle doesn't support text), add the class 'badge-pill' or 'badge-circle' in addition to the colour.

<div class='row'>

  <h5>Informative Example</h5>

  <div class='badge-main badge-pill'>New</div>

</div>

Modals

The modal to the left can be created with the code below. Modals should be top-level but buttons can be anywhere. The button should have a property called onclick="openmodal('yourModalID)". Your modal should have that ID so it can be correctly called. Inside a div has the modal-content class. A span inside should have the close class and optionally our x-button class. It should also say onclick="closemodal('yourModalID')".

The modal opens by default near the top of the screen, which can be changed by using the 'm-middle' class on the outermost modal div.

<div class='button-main px-2' onclick="openModal('basicModal')">Another Button</div>

<div id='basicModal' class='modal m-middle'>

  <div class='modal-content'>

    <span class='x-button close' onclick="closeModal('basicModal')">X</span>

    <h3 class='border-bottom'>Basic Modal</h3>

    <p>This modal is an example.</p>

  </div>

</div>

Accordion

Accordions can either allow only one item open at once or allow any in any combination. If you include 'acc-only-one' as a class on the buttons, that accordion will only allow one to be open at once, which will not affect any accordions lacking these classes. The button contains the text as the header and the panel contains anything you want inside.

The code example is a part of the shown example.

<div class='accordion-holder'>

  <button class='accordion'>First</button>

  <div class='panel'>

    <p>Text...</p>

  </div>

  <button class='accordion'>Second</button>

  <div class='panel'>

    <p>Text...</p>

  </div>

</div>
